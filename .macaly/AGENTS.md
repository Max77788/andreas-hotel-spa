## Design System

**Aesthetic:** Desert luxury / California boutique hotel — inspired by 1 Hotels SF (earth tones, minimal, elegant)
**Fonts:** Cormorant Garamond (display/headings) + DM Sans (body)
**Color tokens:**
- `--hotel-cream: #f5f0e8` (background)
- `--hotel-sand: #e8ddd0` (secondary bg)
- `--hotel-charcoal: #2a2118` (dark text/backgrounds)
- `--hotel-terracotta: #b8743d` (accent/CTAs)
- `--hotel-gold: #c9a96e` (gold highlights)

**Typography conventions:** All-caps tracking-widest labels, Cormorant Garamond light for headings, DM Sans for body
**Button style:** No border-radius (radius: 0), border + hover fill pattern
**Nav style:** Fixed transparent → charcoal on scroll, centered logo, hamburger left, Book Now right
